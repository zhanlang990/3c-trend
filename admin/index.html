GitHub 自动同步设置

配置GitHub Token后，点击"保存配置"将自动提交到GitHub仓库，爬虫第二天运行时自动读取最新配置，无需手动操作。

已预填默认仓库地址，通常无需修改
Token仅保存在浏览器localStorage中，不会上传到任何服务器。需要repo权限以提交文件。
生成方式：GitHub → Settings → Developer settings → Personal access tokens → Generate new token (classic)，勾选 repo 权限
